validol by kmerenkov

Data structures (e.g. JSON) validator for Python language. Module that makes json validation simple.

Features

  • straightforward to write schemes
  • simple
  • fast
  • no third-party dependencies
  • many unit-tests

Example

 1 >>> scheme = {"status": lambda s: s in ["ok", "error"],
 2               "body": str}
 3 >>> data = {"status": "ok", "body": "Been there done that"}
 4 >>> validate(scheme, data)
 5 True
 6
 7 >>> scheme = {"status": lambda s: s in ["ok", "error"],
 8               "body": str}
 9 >>> data = {"status": "enonsense", "body": "Gotcha!"}
10 >>> validate(scheme, data)
11 False

* Used vim's TOhtml function

Dependencies

Python itself (tested on 2.5 and 2.6)

Install

sudo python setup.py install
